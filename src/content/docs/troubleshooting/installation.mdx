# Installation Problems

Fix common issues when setting up Runtipi.

## Overview

- Time: 5–15 minutes
- Difficulty: Beginner
- Prerequisites: Server access and basic Docker knowledge

## Symptoms

- Docker not found or wrong version
- Compose fails to start services
- Permission denied writing to data directories

## Quick fixes

- Verify Docker/Compose are installed and versions are supported
- Check filesystem permissions for the Runtipi directories
- Ensure required ports (80/443) are free

## Deeper checks

- Inspect logs: `docker compose logs --tail=200`
- Validate config: `docker compose config`
- Test DNS and outbound network access
