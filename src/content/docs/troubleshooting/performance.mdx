# Performance Issues

Find and fix slowdowns in your Runtipi environment.

## Overview

- Time: 10–30 minutes
- Difficulty: Intermediate
- Prerequisites: Access to host metrics and container logs

## Symptoms

- Slow page loads or timeouts
- High CPU/RAM or disk IO
- Spikes during backups or updates

## Quick wins

- Restart only the affected services
- Limit noisy logs and prune unused images/containers
- Ensure the host has enough free disk space

## Investigation

- Host metrics: `top`, `htop`, `iostat`, `df -h`
- Container stats: `docker stats`
- Database health: check slow queries and memory usage

## Long-term improvements

- Use SSD storage and adequate RAM/CPU
- Enable caching where supported by apps
- Schedule backups/updates in off-peak hours
